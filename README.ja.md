# PaperAgent

PaperAgent はローカルファーストの論文執筆ワークスペースです。ブラウザで LaTeX とコードを編集し、バックエンドで Python/R を実行して LaTeX をコンパイルし、PDF とログを一か所で確認できます。

## ビジョン

PaperAgent は研究の雑務から人を解放し、「Only Ideas」に戻すために作られています。
目的はシンプルです。考えることは人に、繰り返しはシステムに。
あなたはアイデアと物語に集中し、PaperAgent が実行ループを回します。

## フィロソフィー

- ローカルファースト・プライバシーファースト：データと実行は基本的に端末内。
- アイデアファースト：コンセプトから実行可能な論文まで摩擦を最小化。
- 小さく可逆：変更は透明で戻せる。
- ツールは仕事を減らす：自動化は手間を削るため。

## ロジック（どう動くか）

1) チャット → 編集：要望を伝えると、必要なファイルが更新されます。
2) 実行 → コンパイル：Python/R を実行し、LaTeX をコンパイルして図表を生成。
3) プレビュー → 反復：PDF とログを確認し、素早く修正して繰り返す。

## 得られるもの

- Docker サンドボックスに接続した Web 端末
- LaTeX プロジェクトの雛形とワンクリックコンパイル
- 図表・実験のための Python/R 実行
- PDF プレビューとログ
- シンプルでミニマルな PWA UI
