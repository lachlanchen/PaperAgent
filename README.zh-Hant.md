# PaperAgent

PaperAgent 是一個本地優先的論文寫作工作台：在瀏覽器中編輯 LaTeX 與程式碼，後端執行 Python/R 與 LaTeX 編譯，並在同一處預覽 PDF 與編譯日誌。
