# PaperAgent

PaperAgent là không gian làm việc viết bài nghiên cứu theo hướng local-first: chỉnh sửa LaTeX và mã trong trình duyệt, chạy Python/R và biên dịch LaTeX ở backend, và xem PDF cùng log tại một nơi.
